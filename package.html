Package Design Gallery

This page features three package design projects. The first project is a redesigned book jacket for the novel, "Lord of the Rings. The second is the creation of the brand Carpe Natem, which is an online underwear delivery service. The large box features the brand name, and the smaller individualized boxes feature the type of underwear that will be included, along with a quirky saying. The black silhouette represents a window cutout of the style of underwear, so that the consumer can see inside of the box. The last project is a 20 page magazine spread dedicated to wedding styles. This page specifically features the magazine cover, as well as a few spreads from inside of the magazine.
